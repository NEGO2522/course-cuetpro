# CUET (DU) Seats & Cutoffs Explorer

A fast, responsive React application designed to help students explore undergraduate programs, colleges, seat allocations, and historical cutoffs for Delhi University. 

This tool takes raw admission data and presents it in a clean, user-friendly interface, allowing applicants to make informed decisions easily.

## Features

- **Program-Centric View**: Data is organized by academic programs rather than individual colleges for easier browsing.
- **Advanced Filtering**: Filter programs by academic categories (e.g., Languages, Arts & Social Sciences, Science, etc.).
- **Search & Sort**: Quickly search for specific programs and sort them alphabetically, by total seats, or by top cutoffs.
- **Detailed College Insights**: Click on any program to view a detailed modal containing college-wise seat matrices and cutoff scores across all social categories (UR, OBC, SC, ST, EWS, PwBD).
- **Responsive Design**: Fully mobile-responsive interface built with modern CSS variables and a clean grid layout.

## Tech Stack

- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS (Responsive, CSS Variables, Flexbox/Grid)
- **Data Processing**: Python (for parsing and transforming raw PDF/Text data into structured JSON)

## Getting Started

To run this project locally, follow these steps:

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/NEGO2522/course-cuetpro.git
   cd course-cuetpro/course-tool
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`.

## Data Processing

The application consumes structured JSON data located in `src/data/colleges.json`. This data is generated by running Python scripts (`extract_pdfs.py` and `parse_data.py`) against raw university PDF records to clean, format, and aggregate the statistics automatically.
